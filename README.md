# StockBot
Discord bot to provide stock information

## Commands
### price
#### format:
!stock price \<Ticker\>
#### returns:
Company name, ticker, price, day change, % day change

## How to use
Open '.env_example', insert your discord bot API key and rename to '.env'

## Dependencies
### Python 3.10

**Python Site:** https://www.python.org/downloads/
